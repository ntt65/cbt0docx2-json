# 📝 가스기능사 CBT JSON 오탈자 교정 이력

이 문서는 제공된 가스기능사 학습 자료 및 표준 가스안전기준(KGS Code)을 기반으로, `data/gas/gas_questions.json` 데이터 내의 OCR 인식 오류, 오탈자, 한글 깨짐 등을 정교하게 교정하여 데이터 무결성을 확보한 내역을 기록하는 공식 정오표(Errata)입니다.
